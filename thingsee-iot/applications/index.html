Common Features

All Thingsee mobile applications share these capabilities:

  • QR Code Scanning - Add devices by scanning their QR codes
  • Device Identification - View device info and status
  • Deployment Groups - Organize devices into logical groups
  • Export - Generate CSV files for documentation
