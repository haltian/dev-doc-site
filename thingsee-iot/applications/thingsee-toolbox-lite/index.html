Thingsee Toolbox Lite

Simplified mobile application for installation and maintenance tasks

Thingsee Toolbox Lite is a streamlined mobile application for installation and maintenance purposes. It provides essential functionality for managing Thingsee IoT devices with a simplified interface.

Download

  • App Store (iOS)
  • Google Play (Android)

Requirements: iOS 12+ or Android 10+

Internet Required

The application requires an internet connection to work correctly.

Getting Started

Prerequisites

  1. Download Thingsee Toolbox Lite
  2. Have your stack QR codes ready
  3. Grant camera and storage permissions

Adding a Stack

  1. Tap Add Stack
  2. Scan QR codes in order
  3. Press Save after stack information appears

Removing a Stack

  1. Open menu (top right corner)
  2. Select Remove
  3. Confirm or cancel

Sessions

Sessions are time-limited work periods for installations or maintenance.

Creating a Session

  1. Tap + to create new session
  2. Enter:
    • Title - Session name
    • Description - Work being performed
    • Deployment group - Target group for devices
Session Limits
  • Sessions are open for 20 hours from creation
  • Creating a new session closes the previous one
  • Only one session can be active at a time

Editing a Session

  1. Open the active session
  2. Tap menu → Edit
  3. Modify name or description
  4. Save changes

Exporting Session Data

After completing work:

  1. Open the session
  2. Tap menu → Export log
  3. Choose sharing method

Ended sessions can be reopened to export CSV files.

Device Management

Check Device

Look up any device without adding it to a session:

  1. Open menu → Check device
  2. Scan QR code OR enter the TUID
  3. View device info and recent events

Adding Devices

  1. Open a session
  2. Tap +
  3. Scan the device QR code
  4. Device appears in:
    • Devices list - All added devices
    • Log list - Installation record

Editing Devices

Devices must be in an active session to edit:

  1. Select the device
  2. Tap menu → Edit
  3. Modify configurations and descriptions
  4. Save Changes

Configurable Devices

Configuration is available for:

  • Thingsee PRESENCE
  • Thingsee ENVIRONMENT
  • Thingsee COUNT
  • Thingsee BEAM
  • Haltian RADAR

Replacing Devices

For maintenance visits requiring device replacement:

Method 1: From session main view

  1. Open menu → Replace device
  2. Scan original device QR code
  3. Scan new device QR code
  4. Configure and add installation note
  5. Tap Replace device

Method 2: From device view

  1. Open the device (must be in current session)
  2. Tap menu → Replace
  3. Scan new device QR code
  4. Configure and save

After replacement:

  • New device appears in Devices list
  • Old device with replacement info appears in Log
  • Replaced devices remain in CSV export

Removing Devices

  1. Add device to session with +
  2. Scan QR code or enter TUID
  3. Open device in app
  4. Tap menu → Remove
  5. Confirm removal

Export

CSV Export

Export installation/maintenance records as CSV:

  1. Open the session
  2. Tap menu → Export log
  3. Choose sharing method

CSV includes:

  • Device TUIDs
  • Descriptions
  • Timestamps
  • Actions performed (add, replace, remove)
  • Configuration changes

Workflow Examples

Installation Workflow

  1. Create session for deployment group
  2. Add devices by scanning QR codes
  3. Configure each device as needed
  4. Export CSV when complete
  5. Remove session

Maintenance Workflow

  1. Create session for deployment group
  2. Check device status using Check device
  3. Add devices requiring work to session
  4. Replace or remove as needed
  5. Export CSV for records
  6. Remove session
