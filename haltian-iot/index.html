Haltian IoT

Haltian IoT platform documentation - next-generation IoT connectivity, APIs, and cloud services

Overview

Haltian IoT is Haltian’s next-generation IoT platform, providing secure, scalable device connectivity and data services with modern APIs and enhanced capabilities.

Platform Highlights

  • GraphQL Service API - Modern, flexible API for device management and data access
  • MQTT Stream API - Real-time data streaming with structured topics
  • Data API (Parquet) - Bulk data export for analytics and data lakes
  • Role-Based Access Control - Fine-grained permissions with organizational scopes
  • Haltian IoT Studio - Web application for device management and visualization
