Thingsee ANGLE

Angle orientation sensor for measuring rotational position of valves and handles (EOL)
End-of-Life Product

Thingsee ANGLE has been discontinued and is no longer available for purchase. This documentation is provided for existing deployments only.

Overview

Thingsee ANGLE was an orientation sensor designed to measure angular position in degrees. When attached to manual valve handles, levers, or other rotating equipment, it reported the exact rotational position - enabling remote monitoring of valve states without requiring valve replacement or complex instrumentation.

Key Features

  • Angle measurement - Reports orientation in degrees
  • Simple attachment - Mounts directly to existing valve handles
  • Non-invasive - No need to replace or modify existing equipment
  • Wireless reporting - Real-time position data via mesh network

Primary Use Cases

  • Manual valve position monitoring
  • Lever and handle orientation tracking
  • Equipment state monitoring (open/closed/partial)
