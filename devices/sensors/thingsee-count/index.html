Thingsee COUNT

Time-of-flight people counting sensor for doorways and passages

Overview

Thingsee COUNT is a time-of-flight (ToF) based people counting sensor designed for accurate visitor counting and utilization monitoring. Installed above doorways or passages, it detects movement passing beneath the device and determines directional flow (entries vs. exits).

Unlike PIR-based sensors, Thingsee COUNT uses laser-based distance measurement to detect people, providing higher accuracy even in challenging lighting conditions and with simultaneous passages.

Key Features

Time-of-Flight Technology

  • Laser distance measurement - Precise detection regardless of lighting
  • Directional counting - Distinguishes entries from exits
  • Higher accuracy - Works with simultaneous passage (2-3 people side-by-side)
  • Wide detection range - Supports passageways up to 1000mm width

Powered Operation

  • USB-powered - External power via USB connector (no battery limitations)
  • Continuous operation - 24/7 monitoring without battery concerns
  • Stable performance - Consistent power eliminates battery-related accuracy degradation

Installation Flexibility

  • Cradle mounting system - Secure sensor housing with cable strain relief
  • Two mounting methods - Tape or screw installation
  • Height adjustment - 230cm recommended (supports various ceiling heights)

Use Cases

Typical applications for Thingsee COUNT:

  • Visitor counting - Retail stores, museums, libraries
  • Meeting room utilization - Office space optimization
  • Building traffic analysis - Lobby and corridor monitoring
  • Occupancy tracking - Real-time space usage monitoring

The device determines movement direction during installation configuration using the Thingsee Toolbox application, so it knows which side represents entering versus exiting a space.

Quick Start

Installation steps:

  1. Select installation location (above doorway, centered, 230cm height)
  2. Mount cradle to ceiling/wall (tape or screw method)
  3. Install USB cable through cradle holder
  4. Attach sensor unit to cradle
  5. Configure direction using Thingsee Toolbox app (scan QR code)
  6. Connect USB power supply

Requirements:

  • Maximum passageway width: 1000mm
  • Maximum passageway height: 2100mm
  • Installation height: 230cm recommended
  • Power supply: USB 5V (included)
  • Network: Thingsee Gateway within 20m
Power Dependency

Unlike battery-powered sensors, Thingsee COUNT requires continuous USB power. If power is cut off, the sensor’s counter will reset to zero.

Important Considerations

Lighting Sensitivity

⚠️ Avoid bright sunlight or incandescent/fluorescent/halogen lamps:

  • Sensor accuracy affected by prolonged light exposure
  • Install in areas without direct sunlight or bright artificial lighting
  • For optimal functionality, choose locations with moderate lighting

Power Continuity

⚠️ Counter resets if power lost:

  • Sensor requires continuous USB power
  • If electricity cut off, counter resets to zero
  • Use reliable power source (avoid outlets on switchable circuits)
  • Consider UPS (Uninterruptible Power Supply) for critical applications

Network Coverage

  • Maximum 20m distance from Thingsee Gateway or another sensor
  • Wirepas mesh network automatically routes data
  • Plan sensor placement to maintain network coverage
