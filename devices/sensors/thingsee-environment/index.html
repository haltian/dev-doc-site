Thingsee ENVIRONMENT

Multipurpose environmental sensor with temperature, humidity, pressure, light, accelerometer, and magnetic switch for versatile monitoring applications

Overview

Thingsee ENVIRONMENT is a versatile, battery-powered wireless IoT sensor designed for comprehensive environmental monitoring across multiple business verticals. Combining six sensor types in one compact device, it provides extensive monitoring capabilities from basic climate control to advanced machine usage tracking.

Key Features

  • 6-in-1 sensor - Temperature, humidity, pressure, light, motion, magnetic switch
  • Long battery life - Up to 5 years (standard mode), 4 years (Machine Usage Mode)
  • Versatile applications - Climate monitoring, door/window status, machine usage, lighting control
  • Easy installation - Preconfigured from factory, no on-site setup needed
  • Multiple mounting options - Tape, screw, or magnet mount
  • Event-driven reporting - Intelligent threshold-based updates
  • Wide certifications - CE, FCC, ISED, RoHS, RCM, MIC, UKCA, NOM, NYCE, IFT compliant

Operating Modes

Standard Mode (Default)

Best for: Climate monitoring, door/window status, general environmental tracking

  • Measurement interval: Every 5 minutes
  • Reporting: Event-based (threshold changes) + 1-hour heartbeat
  • Battery life: Up to 5 years
  • Applications: Most use cases

Machine Usage Mode

Best for: Industrial equipment monitoring, vibration-based usage detection

  • Measurement frequency: 50Hz (high-speed sampling)
  • Reporting interval: 1 minute (when activity detected)
  • Vibration range: 128mG - 8G/16G
  • Battery life: Up to 4 years
  • Applications: Manufacturing equipment, pumps, motors, conveyors

Configuration: Contact support@haltian.com to enable Machine Usage Mode
