Installation

Physical installation guide for Thingsee ENVIRONMENT including placement strategies, mounting methods, and battery setup for all sensor types

Video Tutorial

Before Installation

Prerequisites

  • Ensure Thingsee Gateway is installed and operational (LED blinking green)
  • Plan sensor locations considering all measurement types needed
  • Prepare installation surface (clean and dry)
  • Select appropriate mounting method for use case

Network Coverage Planning

  • Maximum distance: 20m between sensors or to gateway
  • Use intermediate sensors as routers for distances > 20m
  • Account for signal attenuation from:
    • Fire doors and thick walls
    • Metal structures and equipment
    • Dense building materials

Use Case Identification

Determine which sensors you’ll use:

  • Temperature/Humidity - Climate monitoring
  • Barometric Pressure - Weather tracking, floor detection
  • Ambient Light - Occupancy inference, lighting status
  • Accelerometer - Vibration/motion detection, position sensing
  • Magnetic Switch - Door/window open/close status

This determines optimal placement and mounting method.

Placement Guidelines by Use Case

For Climate Monitoring (Temperature/Humidity)

Recommended Height

  • Standard rooms: 1.5-1.8m from floor (breathing height)
  • Large spaces: Multiple sensors at consistent height
  • Cold storage: Lower placement acceptable (monitor product level)

Good Placement

✅ Representative of room conditions
✅ Central location or along walls
✅ Away from direct airflow
✅ Accessible for battery replacement

Avoid

❌ Direct sunlight or near windows
❌ Near heating/cooling sources or vents
❌ Corners (poor air circulation)
❌ Behind doors or furniture

For Light Sensing (Ambient Light)

Sensor Orientation

  • Sensor face should point toward general room lighting
  • Not obstructed by furniture or equipment
  • Consider multiple light sources

Good Placement

✅ Central ceiling mount (facing down)
✅ Wall mount facing room interior
✅ Position with clear view of lights

Avoid

❌ In shadows or under overhangs
❌ Facing windows (sun overwhelms room lights)
❌ Behind obstacles
❌ In cabinets or enclosed spaces

Note: Light measurement is indicative (not calibrated). Suitable for on/off detection and relative changes, not absolute lux measurement.

For Door/Window Monitoring (Magnetic Switch)

Installation Configuration

Sensor and magnet must be aligned when door/window is closed.

Recommended spacing: 5-15mm gap between sensor and magnet

Door Frame Installation

  1. Sensor location: On door frame (stationary)
  2. Magnet location: On door (moving part)
  3. Alignment: Sensor and magnet parallel, facing each other
  4. Test: Close door, verify alignment before final mounting

Window Installation

  1. Sensor: On window frame
  2. Magnet: On sliding/opening window panel
  3. Check alignment when window closed

Magnet Types

  • Included magnet: Small rectangular magnet (usually provided)
  • Alternative: Any strong permanent magnet (neodymium recommended)
  • Orientation: North/south pole facing sensor (test orientation for best detection)

For Machine Usage Monitoring (Accelerometer)

Mounting Requirements

Critical: Sensor must be mechanically coupled to equipment for vibration detection

Good Practice

✅ Directly on vibrating surface
✅ Rigid mounting (screw preferred over tape)
✅ Flat, clean mounting surface
✅ Consistent sensor orientation
✅ Test detection before finalizing installation

Avoid

❌ Flexible or dampened mounting
❌ Tape mounting on rough surfaces (use screws)
❌ Indirect coupling (e.g., on nearby shelf)
❌ Loose or wobbling mount

Orientation Considerations

  • X, Y, Z axes report separately
  • Consistent orientation across similar equipment
  • Document orientation for data interpretation
  • Most sensitive to vibration perpendicular to mounting surface

Vibration Range Selection

  • Machine Usage Mode: 0-8G (typical machinery)
  • High Movement Mode: 0-16G (heavy machinery, impact loads)
  • Contact Haltian support to configure mode

For Position/Orientation Detection (Accelerometer)

Use Case Examples

  • Detect if cabinet door opened (orientation change)
  • Monitor if equipment moved or tilted
  • Track asset position (vertical vs. horizontal)

Installation Tips

  • Note initial orientation (X, Y, Z gravity values)
  • Secure mounting to asset being monitored
  • Document expected orientations for different states

Mounting Methods

Thingsee ENVIRONMENT supports three mounting methods:

Method 1: Adhesive Tape Mounting

Best for:

  • Smooth, non-porous surfaces (painted walls, tiles, glass, metal)
  • Temporary installations or pilot projects
  • Climate monitoring applications
  • Locations where drilling not permitted

Not suitable for:

  • Rough or textured surfaces
  • High-vibration applications (use screws)
  • Outdoor or high-humidity environments

Materials Needed

  • Double-sided adhesive tape (foam tape recommended)
  • IPA solvent (isopropyl alcohol)
  • Clean cloth

Installation Steps

  1. Clean the surface:

    • Wipe with IPA solvent
    • Allow to dry completely (1-2 minutes)
    • Ensure no dust, grease, or residue

  2. Prepare the sensor:

    • Open sensor with provided opening key
    • Clean sensor back surface (if needed)
    • Locate mounting area on sensor back

  3. Apply tape:

    • Cut tape to match sensor footprint
    • Press tape firmly on sensor back for 5 seconds
    • Remove tape backing paper

  4. Mount the sensor:

    • Position at desired location
    • Press firmly against surface for 10 seconds
    • Hold steady to ensure adhesion

  5. Wait before loading:

    • Allow 15 minutes for adhesive to set
    • Do not disturb during this period
    • Then proceed with battery installation

Method 2: Screw Mounting

Best for:

  • Permanent installations
  • Vibration monitoring applications (Machine Usage Mode)
  • Rough or textured surfaces
  • High-reliability requirements

Required for:

  • Machine usage monitoring (vibration detection)
  • High-vibration environments

Materials Needed

  • 2x screws (size: 2-3mm diameter, length based on wall type)
  • Screwdriver
  • Wall anchors (for drywall or concrete)
  • Drill (if using anchors)
  • Pencil for marking

Installation Steps

  1. Mark screw positions:

    • Hold sensor at desired location
    • Mark screw hole positions with pencil
    • Verify height and alignment

  2. Prepare the wall (if needed):

    • For concrete: Drill pilot holes, insert anchors
    • For drywall: Insert appropriate wall anchors
    • For wood: No anchor needed (pilot hole recommended)

  3. Mount the sensor:

    • Align sensor screw holes with wall holes/anchors
    • Insert screws through sensor mounting holes
    • Tighten securely but do not over-tighten
    • Verify sensor is level and flush with surface

  4. Check stability:

    • Gently pull on sensor to verify secure mounting
    • No wobbling or movement
    • For vibration monitoring: Ensure very rigid connection

Method 3: Magnetic Mounting

Best for:

  • Temporary monitoring or testing
  • Metal surfaces (equipment housings, cabinets, doors)
  • Repositionable installations
  • Quick deployment scenarios

Requirements:

  • Metal surface (ferromagnetic: steel, iron)
  • Magnetic mounting accessory (contact Haltian for availability)

Installation Steps

  1. Prepare surface:

    • Clean metal surface (remove dirt, grease)
    • Ensure flat, smooth area for magnet contact

  2. Attach magnet to sensor:

    • Secure magnetic mount to sensor
    • Verify solid attachment

  3. Position on surface:

    • Place sensor on metal surface
    • Magnet should hold firmly
    • Test: Gently pull to ensure secure attachment

  4. Verify stability:

    • Should not slide or fall when touched
    • If loose, relocate to better surface area
    • Not suitable for high-vibration monitoring

Note: Magnetic mounting may interfere with magnetic switch sensor if used for door/window monitoring. Use screw or tape mounting for magnetic switch applications.

Battery Installation

Battery Specifications

  • Type: 2x AAA alkaline batteries
  • Recommended: Varta Industrial Pro 1.5V AAA LR03
  • Alternatives: High-quality alkaline AAA batteries (Duracell, Energizer)
  • NOT compatible: Lithium, rechargeable NiMH (voltage mismatch)

Installation Steps

For New Installation (Before Mounting)

  1. Open the sensor:

    • Hold sensor in hand
    • Insert opening key in side hole
    • Twist to separate A-cover from B-cover

  2. Install batteries:

    • Insert batteries into A-cover battery compartment
    • Observe correct polarity (+ and - markings inside)
    • Ensure both batteries fully seated

  3. Verify operation:

    • LED will flash briefly to indicate power-on
    • Sensor begins initialization sequence

  4. Close the sensor:

    • Align A-cover with B-cover (Thingsee logo upright)
    • Press covers together until they click securely
    • Verify no gaps between covers

For Wall-Mounted Sensor (Battery Replacement)

  1. Open while mounted:

    • Insert opening key in side hole (sensor remains on wall)
    • Twist to release A-cover
    • Pull A-cover away from B-cover

  2. Remove old batteries:

    • Note battery orientation before removal
    • Remove both batteries
    • Dispose of properly (battery recycling)

  3. Install new batteries:

    • Insert fresh AAA batteries
    • Match polarity markings
    • Ensure full contact

  4. Reattach A-cover:

    • Align covers carefully
    • Press together until secure click
    • Verify Thingsee logo orientation (upright)

Specialized Installation Configurations

Door/Window Monitoring Setup

Complete Installation

  1. Install sensor on door frame (stationary part):

    • Use screw mounting for permanent installation
    • Tape mounting acceptable for temporary setups
    • Position so sensor face can align with magnet

  2. Install magnet on door (moving part):

    • Adhesive-backed magnet typically provided
    • Position to align with sensor when door closed
    • Verify 5-15mm gap when closed

  3. Test alignment:

    • Close door/window
    • Magnet and sensor should be parallel, facing each other
    • Open and close several times
    • Verify IoT Studio shows state changes

  4. Fine-tune positioning:

    • Adjust magnet position if needed
    • Optimal gap: 5-15mm
    • Test detection reliability

Machine Usage Monitoring Setup

Critical Installation Steps

  1. Select mounting location:

    • Flat surface on equipment housing
    • Position that receives vibration clearly
    • Avoid dampened or isolated spots

  2. Use screw mounting (required):

    • Tape mounting insufficient for vibration detection
    • Rigid mechanical coupling essential
    • 2-screw mounting recommended

  3. Verify orientation:

    • Document X, Y, Z axis orientation
    • Consistent across multiple machines
    • Aids in data interpretation

  4. Enable Machine Usage Mode:

    • Contact support@haltian.com
    • Provide sensor ID
    • Specify vibration range needed (8G or 16G)
    • Support will configure remotely

  5. Test detection:

    • Run equipment through normal cycle
    • Verify vibration detection in IoT Studio
    • Adjust sensitivity if needed (contact support)

Post-Installation Verification

Network Connection

  1. Wait for network join:

    • Device automatically joins Wirepas mesh network
    • Typical join time: 1-5 minutes
    • May take up to 10 minutes in busy networks

  2. Verify in Haltian IoT Studio:

    • Log into IoT Studio portal
    • Navigate to device list
    • Confirm sensor appears with correct ID
    • Check “Last Seen” timestamp (should be recent)

Initial Data Collection

  1. First readings:

    • Temperature, humidity, pressure, light available immediately
    • Accelerometer provides orientation data
    • Magnetic switch shows current state
    • Expect first readings within 5-10 minutes

  2. Baseline establishment:

    • Monitor readings during first 24 hours
    • Verify measurements reasonable for environment
    • Check all enabled sensors reporting

Signal Strength Check

  1. Review RSSI values in IoT Studio:

    • Good signal: > -70 dBm
    • Acceptable: -70 to -85 dBm
    • Poor: < -85 dBm

  2. If signal is poor:

    • Add intermediate sensor as router
    • Reposition sensor closer to gateway
    • Check for physical obstructions or RF interference

Functional Testing

For Climate Monitoring

  • Verify temperature reads room temperature
  • Check humidity within expected range
  • Pressure should be ~1013 hPa at sea level (adjust for altitude)

For Light Sensing

  • Turn lights on/off, verify lux changes
  • Should see significant difference (thousands of lux)

For Magnetic Switch

  • Open/close door/window
  • Verify state changes appear in IoT Studio within minutes
  • Counter should increment with each state change

For Machine Usage

  • Run equipment through normal operation
  • Verify vibration detected in IoT Studio
  • Check X, Y, Z axis values change during operation

Installation Checklist

  • Gateway operational and connected
  • Installation location planned (use case considered)
  • Surface cleaned and prepared
  • Sensor unpacked and identified (QR code noted)
  • Mounting method selected (tape/screw/magnet)
  • Sensor securely mounted (or batteries installed if mounting first)
  • Batteries installed correctly (polarity matched)
  • Covers attached securely (logo upright)
  • Device appears in IoT Studio
  • Initial readings received (all active sensors)
  • Signal strength acceptable (RSSI > -85 dBm)
  • Magnetic switch tested (if applicable)
  • Vibration detection verified (if Machine Usage Mode)
  • Documentation updated (location, sensor ID, orientation)
