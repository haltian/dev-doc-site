Thingsee Gateway Global

Plug & play IoT gateway with global LTE Cat M1 and 2G cellular support

Overview

Thingsee Gateway Global is a plug & play IoT gateway for large-scale sensor deployments with worldwide cellular connectivity. Built for ease of deployment, it provides LTE Cat M1 and 2G connectivity out of the box, complete with pre-installed SIM card and managed connectivity subscription.

The gateway connects Wirepas mesh networks of wireless sensors to the Haltian Operations Cloud, supporting deployments from a few sensors to hundreds of devices. Its plug & play design eliminates complex network configuration - simply power on and sensors automatically join the network.

Key Features

  • Global Cellular Connectivity: LTE Cat M1 with 2G fallback, worldwide coverage
  • Plug & Play: Pre-configured SIM with managed connectivity subscription
  • Wirepas Mesh Support: BLE mesh network for up to 200-500 sensors
  • USB Powered: Simple micro USB power (5V)
  • Compact Design: Portable, desktop or wall-mountable
  • Status LEDs: Visual indication of power, cellular, and mesh status
  • Cellular Metrics: RSSI, RSRP, RSRQ, SINR monitoring
  • Automatic Network Formation: Self-organizing sensor mesh

Quick Start

  1. Insert SIM card and power on - Use included SIM, connect micro USB
  2. Verify cellular connection - Check LED status
  3. Deploy sensors - Position sensors within range
  4. Monitor in Studio - Verify gateway and sensors in Haltian IoT Studio

Network Structure

Haltian devices build self-organizing mesh networks automatically:

Key Network Characteristics:

  • Self-Organizing: Sensors and gateways form optimal routes automatically
  • Multi-Hop: Sensors relay data through neighbors to reach gateway
  • Resilient: Network adapts to changing conditions and failures
  • Secure: Closed network, no third-party device access
  • Scalable: Supports few to hundreds of sensors per gateway
